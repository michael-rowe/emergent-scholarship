**Version:** 0.1
**Status:** Scaffolding — this document defines its own purpose before attempting to fulfil it.

## What this document is

This is an articulation of the intellectual throughline running across my work — not a personal profile, not a set of instructions for AI tools, and not a mission statement. It exists to give coherence to work that spans multiple projects, contexts, and outputs.

It answers the question: *what is all of this for?*

## What this document is not

- **Not a personality profile.** My values, preferences, and dispositions are captured elsewhere — in memory systems, user preferences, and the writing itself. This document isn't about me.
- **Not a system prompt.** It doesn't instruct Claude or any other tool how to behave. It provides context that tools can draw on, but it doesn't direct them.
- **Not a manifesto.** It isn't trying to persuade anyone of anything. It's a private working document that keeps the work honest.

## How this document should be used

When working across projects — writing, presenting, building, thinking — this document serves as a reference point for coherence. Not consistency (the work should evolve and contradict itself where necessary), but coherence: the sense that disparate outputs are part of a larger enquiry.

It should be consulted when:

- Starting something new, to ask whether it connects to the throughline or departs from it (both are fine, but the departure should be conscious)
- Finishing something, to ask whether it moved the enquiry forward or merely performed productivity
- Feeling lost in a project, to reconnect with what the work is actually trying to do

## How this document should evolve

This is a living document. It should be revised when the throughline shifts — not when the wording feels imperfect, but when the underlying enquiry genuinely changes direction. Discomfort with what's written here is a signal to interrogate the work, not to edit the file.

Version history should be preserved. Earlier versions aren't wrong — they're the intellectual trail.
